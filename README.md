## ğŸš€ Getting Started

### Prerequisites

Make sure you have the following installed:
- Node.js (18.0.0 or later)
- npm or yarn or pnpm

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/PreetKot/SPEC-UI.git
   cd SPEC-UI
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application.

### Using Components

Simply copy the component code from our website and paste it into your project:

```tsx
import { AnimatedForm } from '@/components/specui/animated-form'

export default function MyPage() {
  return (
    <div>
      <AnimatedForm name="John Doe" />
    </div>
  )
}
```
